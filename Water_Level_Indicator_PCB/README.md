# Water Level Indicator with PCB Design

## Project Overview

This project is a Water Level Indicator circuit designed and implemented as a PCB using Altium Designer. The circuit indicates different water levels in a tank using LED indicators and a buzzer alert.

## Objective

The main objective of this project is to design a simple, low-cost, and compact PCB-based water level indicator system to reduce water wastage and provide clear water level indication.

## Working Principle

The circuit works based on the conductivity of water. Probes are placed at different levels inside the water tank. When water reaches a particular probe, the corresponding circuit path is completed, and the related LED turns ON. When the tank reaches the full level, the buzzer gives an alert.
